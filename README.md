# Climate Change in india effecting human sustainability 

## Project Description

This project uses a climate dataset containing weather and environmental information from different cities in India. The goal is to explore the data, understand weather patterns, and perform basic data analysis using Python and Pandas.

## Dataset

**File:** `climate_dataset.csv`

The dataset contains the following information:

- Date
- City
- State
- Maximum Temperature (°C)
- Minimum Temperature (°C)
- Average Temperature (°C)
- Humidity (%)
- Rainfall (mm)
- Wind Speed (km/h)
- AQI (Air Quality Index)
- AQI Category
- Pressure (hPa)
- Cloud Cover (%)
- Visibility (km)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- VS Code
